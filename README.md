## Purpose

Project Pages is a Krita plugin to Compile files into a single project file.


## Features
* Recent Projects
* List of Pages
* Context Menus
* Image connected to a Note
* Page Template
* Rename Layers
* Guides ( List, Mirror )
* Mirror Fix
* Read ![wiki](https://github.com/EyeOdin/project_pages/wiki) for more information
